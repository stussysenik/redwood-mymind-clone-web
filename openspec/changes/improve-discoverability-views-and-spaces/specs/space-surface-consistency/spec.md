## ADDED Requirements

### Requirement: Spaces share the main browsing language

The system SHALL present spaces with the same discoverability quality as the main library, including clearer metadata, stronger visual hierarchy, and consistent surface styling.

#### Scenario: Spaces index uses polished surface cards

- **WHEN** a user opens the spaces index
- **THEN** each space MUST present name, type, and card-count metadata in a clear, scan-friendly card

#### Scenario: Space detail matches collection-shell controls

- **WHEN** a user opens a specific space
- **THEN** the detail page MUST use the same count/toggle/header conventions as the primary collection browsing surfaces
